# Smart Playlist Recommender System

This folder now contains a clean full-stack version of the PPT project using:

- `frontend/` for the Apple Music inspired UI
- `backend/` for the pure Java backend

## Java concepts mapped from the PPT

- `Collections Framework`: song library, filters, and saved playlists
- `Exception Handling`: invalid login, invalid inputs, duplicate saves
- `Encapsulation`: private model fields with safe access methods
- `Multi-threading`: Java HTTP server executor plus recommendation worker pool

## Run the project

1. Open a terminal in [backend](/Users/jayesh/my-project/backend)
2. Run `./run.sh`
3. Open [http://127.0.0.1:8080](http://127.0.0.1:8080)
